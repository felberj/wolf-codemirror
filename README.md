# wolf-codemirror

Provides [CodeMirror](http://codemirror.net) syntax highlighter for [Wolf CMS](http://www.wolfcms.org) backend editor.

## Support parser:
* HTML
* CSS
* Javascript
* PHP

## Optional

Integrate with [File Manager](http://www.wolfcms.org/repository/8) plugin.
